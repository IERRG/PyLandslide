PyLandslide - landslide susceptibility mapping and uncertainty analysis in Python
===================================================================================

PyLandslide is a machine learning-assisted open-source Python tool for landslide susceptibility mapping and uncertainty analysis.
The source code of PyLandslide is hosted on the following `GitHub Repository <https://github.com/SHESP-Lab/PyLandslide>`_.
